## 要实现的函数清单

在 `include/adc_driver.h` 和 `src/adc_driver.c` 中实现：

1. `adc_init()` — 初始化 ADC 时钟、分辨率、采样时间
2. `adc_read()` — 启动单次转换并等待结果
3. `adc_raw_to_mv()` — ADC 原始值转电压值

## 编译运行

```bash
cd mcu/adc
make all        # 编译
make flash      # 烧录
```
